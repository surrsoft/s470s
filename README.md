# s470s

Расширение для браузера для быстрого копирования часто используемых текстовых заметок.

## Описание

s470s — это простое и удобное расширение для Chrome и Edge, которое позволяет сохранять текстовые заметки и копировать их в буфер обмена одним кликом. Идеально подходит для хранения и быстрого доступа к часто используемым фрагментам текста, таким как email-адреса, шаблоны ответов, фрагменты кода или любой другой текстовой информации.

## Возможности

- **Быстрое копирование** — кликните на заметку, чтобы скопировать её текст в буфер обмена
- **Организация заметок** — перетаскивайте заметки для изменения их порядка (drag & drop)
- **Описания** — добавляйте опциональные описания к заметкам для лучшей организации
- **Редактирование** — легко изменяйте существующие заметки
- **Локальное хранение** — все данные сохраняются локально в браузере
- **Визуальные подсказки** — всплывающее уведомление при копировании

## Установка

### Для разработчиков

1. Склонируйте или скачайте этот репозиторий
2. Откройте Chrome/Edge и перейдите на страницу расширений:
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
3. Включите "Режим разработчика" (Developer mode)
4. Нажмите "Загрузить распакованное расширение" (Load unpacked)
5. Выберите папку с расширением

## Использование

### Добавление заметки

1. Кликните на иконку расширения в панели инструментов браузера
2. Нажмите кнопку `+` в правом верхнем углу
3. Введите текст, который нужно копировать
4. Опционально добавьте описание для удобства поиска
5. Нажмите "Save"

### Копирование текста

Просто кликните на любую заметку — текст автоматически скопируется в буфер обмена. Вы увидите уведомление "Copied!".

### Изменение порядка

Зажмите и перетащите заметку за иконку `⋮⋮` слева для изменения порядка.

### Редактирование и удаление

- Нажмите иконку **✎** (карандаш) для редактирования заметки
- Нажмите иконку **✕** (крестик) для удаления заметки

## Технические детали

- **Manifest Version**: 3
- **Permissions**:
  - `storage` — для сохранения заметок
  - `clipboardWrite` — для копирования в буфер обмена
- **Хранилище**: Chrome Storage API (локальное)

## Разработка

### Требования

- Современный браузер на базе Chromium (Chrome, Edge, Brave и т.д.)

### Локальное тестирование

1. Внесите изменения в код
2. Перезагрузите расширение на странице расширений
3. Откройте всплывающее окно расширения для тестирования

### Функции JavaScript

- `loadNotes()` — загрузка заметок из хранилища
- `saveNotes()` — сохранение заметок в хранилище
- `render()` — отрисовка списка заметок
- `addNote()` — создание новой заметки
- `updateNote()` — обновление существующей заметки
- `deleteNote()` — удаление заметки
- `copyToClipboard()` — копирование текста в буфер обмена
- Функции drag & drop для изменения порядка заметок
